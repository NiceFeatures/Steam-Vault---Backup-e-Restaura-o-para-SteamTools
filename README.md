# Steam-Vault---Backup-e-Restaura-o-para-SteamTools

Um programa em python feito para realizar backups e restaurá-los posteriormente para quem usa o SteamTools.

Atualemnte ele sava as pastas:

-depotcache (arquivos .manifest)
-stplug-in (arquivos .lua)
-appcache\stats (Conquistas basicamente) 
-userdata (Tempo de jogo, Screenshots e possíveis saves)

Este programa foi feito por uma pessoa que não sabe realmente programar, então podem existir bugs.
Estou vende de fazer uma veresão integrada ao Millennium, ai o backup e restauração seriam feitos de dentro da Steam.
